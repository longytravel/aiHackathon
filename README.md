# The Art of Planning - AI Hackathon Session 2

An interactive presentation website showcasing "How I Stopped Fighting AI and Started Shipping."

## Overview

This is the companion website for Session 2 of the AI Hackathon, featuring:

- **The Hero's Journey**: A three-act structure covering the transformation from "I Can't" to "I'm Building"
- **The 5 Things Framework**: What AI actually needs you to plan
- **Live Demo**: The Banking Communication System built with these principles
- **Code Showcase**: Modular architecture examples from a real AI-powered system

## Tech Stack

- **Framework**: Next.js 16 with App Router
- **Styling**: Tailwind CSS with custom Blueprint theme
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Deployment**: Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the presentation.

## Features

- Stunning "Architect's Blueprint" dark theme
- Interactive timeline of the disaster story
- Animated framework cards with hover effects
- Code showcase with syntax highlighting
- Video placeholders ready for content
- Mobile-responsive design
- Blueprint grid background effect
- Cyan/Gold accent color scheme

## The Banking Communication System

This presentation showcases the [Banking Communication System](https://github.com/longytravel/bankingCommunicationSystem) - a modular AI-powered system that transforms standardized bank letters into personalized digital voice notes.
